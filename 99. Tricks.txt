1. Convert your command window code into a live script
    > Open Command History
    > Select the commands
    > Right click
    > Create Live Script

2. Create beautiful docs with this:
    * Split code: %% sectionName
    * Add text
    * ...
    
    * $M_{some_text}$
    * $\sin$, $\cos$
    * $\times$
    * $\alpha$
    * $\beta$
    * $\delta$
    * $\gamma$
    * ...

3. Import data tools (csv, xlsx, ...)
