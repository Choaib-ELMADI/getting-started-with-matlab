1. What is MATLAB:
    MATLAB is an environment for all kinds of technical computing, like:
        * Data analysis
        * Simulation
        * Algorithm development
        * ...

2. Development Environment:
    > Current Folder:   => Show folders and files
    > Command Window:   => Write and execute commands
    > Workspace:        => Show and manipulate variables
    > Command History:  => Show the executed commands
    > Details:          => Show details of a variable
